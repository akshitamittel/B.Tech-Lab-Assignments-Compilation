1. Unzip and go into the directory through command-line.
2. Open a new terminal in the same directoy (Ctrl + Shift + N)
3. Run the server first: python myServer.py (Argument a) (Argument b)
   (Argument a) -> Data loss in percentage (1 - 100)
   (Argument b) -> Average time delay. (in milliseconds) (< 1000)
4. Run the client next in the other terminal: python myClient.py
